# 🐍 Assignment 02: Python Mini Project Challenge

## Objective

Apply the Python concepts learned in Week 2 by building a small working program.

This assignment is designed to help you practice problem-solving, logical thinking, and programming fundamentals using Python.

---

## Choose One Level

### 🟢 Beginner – Calculator

Build a simple calculator program that can perform:

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)

Your program should:

* Take user input
* Display the result clearly
* Handle invalid input gracefully

### Bonus Challenge

Prevent the program from crashing when dividing by zero and display a friendly error message instead.

**Deliverable:** Python code + screenshot of output

---

### 🔵 Intermediate – Grade System

Create a program that accepts marks from three subjects and calculates:

* Total Marks
* Percentage
* Grade
* Remarks

Example:

```text
Total: 240
Percentage: 80%
Grade: B
Remarks: Very Good
```

### Bonus Challenge

Allow the program to process multiple students using a loop.

**Deliverable:** Python code + sample output

---

### 🟠 Advanced – Password Strength Checker

Create a program that checks the strength of a password.

Consider:

* Minimum length
* Uppercase letters
* Numbers
* Special characters

The program should classify passwords as:

* Weak
* Medium
* Strong

### Bonus Challenge

Suggest ways to improve weak passwords.

Example:

```text
Password: hello123

Strength: Medium

Suggestion:
Add an uppercase letter and special character.
```

**Deliverable:** Python code + sample output

---
