## 📝 Assignment

### Assignment 02: Python Mini Project Challenge

Choose **ONE** project based on your confidence and skill level.

| Level           | Project                   |
| --------------- | ------------------------- |
| 🟢 Beginner     | Calculator                |
| 🔵 Intermediate | Grade System              |
| 🟠 Advanced     | Password Strength Checker |

### Project Options

#### 🟢 Beginner – Calculator

Build a calculator that supports:

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)

**Bonus:** Handle division-by-zero gracefully.

---

#### 🔵 Intermediate – Grade System

Create a program that:

* Accepts marks from 3 subjects
* Calculates total and percentage
* Assigns a grade
* Displays remarks

**Bonus:** Support multiple students using loops.

---

#### 🟠 Advanced – Password Strength Checker

Create a program that:

* Checks password length
* Detects uppercase letters
* Detects numbers
* Detects special characters
* Classifies passwords as Weak, Medium, or Strong

**Bonus:** Suggest improvements for weak passwords.
